# YT to Subs

A minimalist, open-source browser extension that instantly redirects the algorithmic YouTube homepage to your Subscriptions feed. Designed to help you stay focused on the content you’ve actively chosen to follow.

### Features
- **Instant Redirect:** Routes you automatically and seamlessly to `/feed/subscriptions`.
- **Toggle Switch:** Easily turn the redirect on or off via a sleek popup UI without disabling the extension.
- **Privacy First:** Only requests required permissions for YouTube domains. No data collection, zero tracking.
- **SPA Aware:** Respects YouTube's internal navigation for a smooth, reload-free experience.
